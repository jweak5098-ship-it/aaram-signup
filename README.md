# Aaram — Join the Waitlist

Professional home massage sign-up page for Metro Vancouver.

## 🚀 Live
Visit: **`https://jweak5098-ship-it.github.io/aaram-signup/`**

## ✨ Features
- 7-step interactive sign-up flow
- Animated background & smooth transitions
- Founding member reward codes (auto-generated)
- FormSpree integration for email collection
- Mobile responsive design
- LocalStorage persistence
- Form validation with shake animation

## 📋 Setup
1. Repository created ✅
2. Enable GitHub Pages:
   - Go to repo Settings → Pages
   - Select "Deploy from a branch"
   - Choose `main` branch
   - Save
3. Your page will be live at `https://jweak5098-ship-it.github.io/aaram-signup/`

## ⚙️ Customization
- **Neighbourhood options**: Edit Step 2 (lines ~240-250 in index.html)
- **FormSpree ID**: Update line 382 if needed (currently `xnjkypzw`)
- **Reward offer**: Modify the offer banner text
- **Price range**: Adjust Step 6 min/max values
- **Stats bar**: Update counter numbers and labels

## 📝 Form Fields Collected
- Name & Email
- Neighbourhood
- Massage frequency
- Pain point
- Service type preference
- Price point
- Extended health benefits
- Booking timeline
- Auto-generated reward code

## 📧 FormSpree Integration
All submissions are sent to your FormSpree account. You'll receive emails for each sign-up at the address linked to your FormSpree account.

Built with vanilla HTML/CSS/JavaScript — **no dependencies**, no build step needed.
